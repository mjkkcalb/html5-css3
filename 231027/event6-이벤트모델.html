MDN 이벤트 참조

이벤트 핸들러 evnet handler

  • 이벤트 발생시 실행되는 코드 모음. 실행되는 함수
  • 특정 요소에 지정된 이벤트가 발생하면 이벤트 리스너에 연결된 이벤트 핸들러 실행

이벤트 리스너 event listener

  • 특정 이벤트에 대해 일어날 동작
  • 보통 이벤트 리스너와 이벤트 핸들러를 합쳐 이벤트 리스너라고 함
  1. click: 클릭했을 때
  2. dblclick: 더블 클릭했을 때
  3. mousemove: 마우스가 움직였을 때
  4. mousedown: 버튼을 눌렀을 때
  5. mouseup: 버튼이 떨어졌을 때
  6. mouseover: 대상 위에 마우스를 올렸을 때. 하위 노드에 영향을 줌
  7. mouseout: 마우스가 벗어났을 때. 하위 노드에 영향을 줌
  8. mouseenter: 마우스를 올렸을 때. 하위 노드에 영향을 주지않음
  9. mouseleave: 마우스가 벗어났을 때. 하위 노드에 영향을 주지않음
